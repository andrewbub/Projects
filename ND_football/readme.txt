Andrew Bub and Harry Gebremedhin

This final project was a collaboration between Harry Gebremedhin and Andrew Bub. 
The gfx library requires x11 to run graphics. Use XQuartz and x forwarding.
Please ideally run the files on a student machine.
Run make file and start the program. 
Press the spacebar to shoot the ball, press 'q' to quit and 'r' to restart. 
Enjoy our Notre Dame themed Game!
